## General info

**WaWa Tabor** is an Android app that shows current location of all buses and trams in Warsaw (Poland).

All data is provided by [https://api.um.warszawa.pl/](https://api.um.warszawa.pl/) - website of Miasto Sto≈Çeczne Warszawy (City of Warsaw).
